# Movie-Performance-Success-Analysis

Project Overview

This project analyzes movie success factors using data visualization techniques.
The goal is to understand how financial, market, and content-related factors influence movie performance and profitability.

The analysis focuses on:

Financial performance (revenue, profit, ROI)

Investment efficiency

Market and genre behavior

Trends over time

The project was implemented using Power BI as the primary visualization tool.
